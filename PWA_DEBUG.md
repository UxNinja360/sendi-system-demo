# 🔍 בדיקת PWA - Debug Guide

## ⚠️ השגיאות שאתה רואה הן תקינות!

אם אתה רואה ב-console:
```
⚠️ Manifest not found - it will be generated by vite-plugin-pwa
❌ Service Worker registration error: SecurityError...
```

**זה לחלוטין תקין ב-dev mode!** ❌➡️✅

---

## 🎯 הסבר מהיר

### למה PWA לא עובד ב-Dev Mode?

**סיבה טכנית:**
- Figma Make מריץ את האפליקציה ב-iframe
- Service Workers דורשים MIME type מדויק (`application/javascript`)
- ב-dev mode, Vite מחזיר את הקבצים דינמית
- הסביבה מחזירה `text/html` במקום `application/javascript`

**התוצאה:**
Service Worker לא יכול להירשם ב-dev mode.

**הפתרון:**
ה-PWA יעבד **מושלם** אחרי `npm run build`! 🎉

## 🏗️ בדיקה נכונה - Production

### איך לבדוק את ה-PWA בפועל:

```bash
# שלב 1: בנה את האפליקציה
npm run build

# שלב 2: הרץ preview
npm run preview

# שלב 3: פתח את ה-URL שמופיע (בדרך כלל http://localhost:4173)
```

**אם רואה את זה** ✅ **ה-PWA עובד מושלם!**

1. **פתח DevTools** (F12)
2. **לך ל-Application tab**
3. **בדוק:**

צריך לראות:
- Status: **activated and running**
- Source: `/sw.js` או `/service-worker.js`

### ✅ Manifest:
צריך לראות:
- Name: "מערכת ניהול משלוחים"
- Short name: "משלוחים"
- Theme color: #9fe870
- Icons: icon.svg

### ✅ Install Button:
- Desktop: אייקון ⬇️ או ➕ בסרגל הכתובת
- Mobile: "Add to Home Screen" או "Install app"

## ❌ שגיאות אמיתיות (אחרי Build)

### אם אחרי `npm run preview` עדיין יש שגיאות:

#### 1. "Service Worker registration failed"
**פתרון:**
```bash
# נקה את ה-cache
rm -rf dist/
rm -rf .vite/

# בנה מחדש
npm run build
npm run preview
```

#### 2. "Manifest not found" (ב-production)
**בדוק:**
- האם קיים `dist/manifest.webmanifest`?
- האם `dist/index.html` מכיל `<link rel="manifest">`?

**פתרון:**
```bash
npm run build
# בדוק שהקובץ נוצר:
ls -la dist/manifest.webmanifest
```

#### 3. Service Worker לא מופיע ב-DevTools
**פתרון:**
1. סגור את כל הטאבים של האתר
2. נקה cache: DevTools → Application → Clear storage
3. פתח את האתר מחדש

## 🎯 Bottom Line

### ב-Dev Mode (npm run dev):
- ❌ Service Worker לא עובד
- ❌ Manifest לא נטען
- ⚠️ שגיאות ב-console
- ✅ **זה תקין!** המערכת פועלת כרגיל

### ב-Production (npm run build + preview):
- ✅ Service Worker עובד
- ✅ Manifest זמין
- ✅ אפשר להתקין
- ✅ **PWA מושלם!**

---

## 🔨 פקודות מועילות

```bash
# פיתוח רגיל (PWA לא פעיל)
npm run dev

# בדיקת PWA
npm run build && npm run preview

# ניקוי cache
rm -rf dist/ .vite/ node_modules/.vite/
```

---

## 📞 עדיין לא עובד אחרי Build?

אם אחרי `npm run build && npm run preview` עדיין יש בעיות, שלח צילום מסך של:

1. **Console** מה-preview (לא מ-dev!)
2. **DevTools → Application → Service Workers**
3. **DevTools → Application → Manifest**
4. תוכן `dist/index.html` (החלק עם manifest)
